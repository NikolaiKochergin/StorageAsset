Sceme managment asset
--------------------------
Данный ассет предназначен для управления игровыми сценами.
В своем составе так же имеет модули для сохранения и отслеживания аналитических данных.


Содержит классы: 
LevelLoader - для управления игровыми сценами в загрузочном уровне.
Префаб с данным скриптом необходимо поместить в игровую сцену. 
Если _isClearDataOnStart имеет значение true, при запуске игры из нулевой сцены происходит очистка сохраненных данных.

GameSceneManager - для управления текущей загрузочной сценой.
Имеет методы для запуска следующей игровой сцены и перезапуска уровня.
В поле _repeatFromLevel устанавливается номер уровня, который будет запущен после завершения последней сцены.

Для платформы WebGL ассет так же выполняет сохранение в удаленном хранилище Yandex, для этого необходимо так же установить
https://github.com/forcepusher/com.agava.yandexmetrica

Для активации сбора аналитики в PlayerSettings -> Other Settings в Sctipting Define Symbols необходимо добавить
YANDEX_GAMES для работы YandexMetrica
APP_METRICA для работы AppMetrica
GAME_ANALYTICS для работы GameAnalytic
после установки и настройки соответствующих SDK.
